# ⏱️ TimeEngine Pro
Zamanı yönetmek ve analiz etmek için tasarlanmış modern, web tabanlı bir araç seti.

### 🚀 Özellikler
* **Geçen Zaman Analizi:** Belirli bir tarihten bugüne geçen süreyi salise hassasiyetiyle ölçer.
* **Kalan Zaman (Geri Sayım):** Hedeflenen bir ana ne kadar kaldığını hesaplar ve süre dolduğunda görsel/işitsel uyarı verir.
* **Profesyonel Kronometre:** Tur (lap) kaydetme özelliği ile hassas ölçüm yapar.
* **Detaylı İstatistikler:** Geçen/kalan süreyi otomatik olarak hafta, ay ve yıl birimlerine çevirir.

### 🛠️ Teknolojiler
* HTML5
* CSS3 (Modern UI/UX)
* Vanilla JavaScript (High Precision)
* Web Audio API (Alarm Sistemi)

### 💻 Kurulum
Herhangi bir kurulum gerektirmez. `index.html` dosyasını tarayıcınızda açmanız yeterlidir.
